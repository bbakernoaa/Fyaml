public subroutine parse_line(line, doc, status, line_num)

Parse a single line of YAML content

Processes a line and updates document structure. Handles: - Indentation - Sequence items - Key-value pairs - Nested structures

@param[in] line Input line @param[in,out] doc Document being built @param[out] status Status code @param[in] line_num Line number of the input line
