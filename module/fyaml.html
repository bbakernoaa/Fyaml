A modern Fortran module for parsing YAML files

This module provides functionality to read and parse YAML files into Fortran data structures. It supports nested dictionaries, sequences, and various data types.

Example:

type(fyaml_doc) :: doc
call doc%load("config.yaml")
value = doc%get("config%nested%key")

Note

Supports strings, integers, reals, booleans, nulls, and sequences @version 1.0.0 @see yaml_parser @see yaml_types
